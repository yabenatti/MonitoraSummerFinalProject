# MonitoraSummerFinalProject

This is the final project used as the support material of the workshop at Monitora Summer 2nd edition. The app goes over the following topics:

* Swift
* XCode
* iOS
* Storyboard 
* Segue
* Constraints
* UITableView
* UINavigationController
